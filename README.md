# Mouse Media

Control volume and media playback directly from your mouse using **Caps Lock** as a media mode toggle.

Turn your mouse into a lightweight media controller without interrupting normal mouse behavior.

![Latest Release Poster](assets/latest-release-poster.png)

## Features

- Caps Lock powered media mode
- Scroll wheel controls volume
- Middle click toggles play/pause
- Side mouse buttons control track navigation
- Normal mouse behavior when Caps Lock is off
- Lightweight and always running in the background

## How It Works

Mouse Media only activates when **Caps Lock is enabled**.

### Media Mode (Caps Lock ON)

| Input | Action |
|---|---|
| Scroll Wheel Up | Volume Up |
| Scroll Wheel Down | Volume Down |
| Middle Click | Play / Pause |
| Side Button 1 (XButton1) | Previous Track |
| Side Button 2 (XButton2) | Next Track |

### Normal Mode (Caps Lock OFF)

All mouse inputs behave normally.

- Scroll wheel scrolls normally
- Middle click works normally
- Side buttons work as configured by your system/software

No interruptions. No remapping unless media mode is active.

## Installation

### Option 1: Download Release (Recommended)

1. Download the latest release from the **Releases** page
2. Extract the ZIP file
3. Run `MouseMedia.exe`

Done.

## Requirements

- Windows
- A mouse with:
  - Scroll wheel
  - Middle click
  - 2 side buttons (recommended)
- No additional setup required

## Usage

1. Launch Mouse Media
2. Enable **Caps Lock**
3. Use mouse controls for media
4. Disable **Caps Lock** to return to normal mouse behavior

## Example Workflow

- Watching videos → enable Caps Lock, adjust volume instantly
- Listening to music → skip tracks from mouse buttons
- Gaming or browsing → disable Caps Lock and regain default controls

## Why Mouse Media?

Most mice already have enough buttons to act as a media controller.

Mouse Media adds that functionality without:
- replacing existing behavior permanently
- requiring keyboard shortcuts
- installing bloated software
